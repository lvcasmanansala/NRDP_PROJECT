Accessing the NRDP PCWP Website:

Important Note: There are various ways to access this system.

Via IIS (As an Administrator)

  • Log-In you account through IIS:
    1. Go to iis.gov.ph and login your account.
    2. On the left navigation bar select NDRPPCWP menu.
    3. Enter your account on the modal/pop-up form and click the confirm button.
    4. Once the login is successful, you will be redirected on the NDRP PCWP Dashbord page.

Via Website (As Client)

  • Go to EMB website emb.gov.ph .
    1. From the menu select the Programs menu and from the sub menu select Environmental Research and Laboratory Services.
    2. Scroll Down, and select the National research and development program for prevention and control of water pollution (database repository) menu hyperlink .
    3. Once you successully selected the National research and development program for prevention and control of water pollution (database repository) menu hyperlink, you will be redirected the the Client/Home page.

Directly (As Administrator)

  • Open your browser and enter https://nrdppcwp.emb.gov.ph/admin/ on your URL search bar.

Directly (As Client)

  • Open your browser and enter https://nrdppcwp.emb.gov.ph/ on your URL search bar.

Menus

Important Note: Menus allows the user to navigate different pages on the system

Menu(Main Menu)

  • The Menu is usually located on the top right side of the page.
  • Menu Lists lists of menu that is currently available on the navigation bar.
    1. Documentation Redirects the user to the documentation/user guide page.
    2. Dashboard Redirects the user to the Dashboard page.
    3. Manage Authors Redirects the user to the authors management page.
    4. Manage Research Redirects the user to the research management page.
    5. Profile Allows the user to navigate more features by viewing the sub menu.
  • Sub Menu are menu(s) underlying from a menu that shows more features.
    1. Settings Redirects the user to the settings page Under Development!
    2. Change Password Redirects the user to the change password page Under Development!
    3. Go to Public Page Redirects the user to the client/research page
    4. Log-out Log-Out the user's current session

Managing Authors:

Important Note: Managing Authors allows the user to enroll different authors on the system that can be later on be added to the researches.

Manage Authors

  • From the menu, select/click the Manage Authors menu (The user will be redirected to the Manage Authors page). Manage Authors Page Parts Describes the functionality of buttons, text field and tables on the page.
    1. Breadcrumbs Bar Located on the upper left corner of the page that shows the level of your current page and enables the user to go back to previous pages.
    2. Table List Contains all the lists of authors enrolled on the system
    3. Search Bar Allows the user to filter authors based on a specific keywords that matches the query of authors.
    4. Pagination The pagination paginates the table of authors by (5) for each page to prevent bulk list of authors that may confuse the user.
    5. Add New Author To add/enroll new author on the system, click the add button on the right most column of the table.
      1. You will be redirected to add new author page. Please ensure that the fields is filled-up with proper details before submitting. Text Fields:
        • Suffix Name (Optional)
        • First Name (Required)
        • Middle Name (Optional)
        • Last Name (Required)
        • Educational(Degree) Prefixes (Optional) e.g: click here .
        Buttons
        • Save Saves the record and redirects to the Manage Authors Page
        • Save and add another Saves the record but retains to the current page
        • Back Redirects to the Manage Authors Page
    6. Edit Author To update/edit existing author on the system, click the edit button on the right most column of the record.
      1. You will be redirected to edit author page. Please ensure that the fields is filled-up with proper details before submitting. Text Fields:
        • Suffix Name (Optional)
        • First Name (Required)
        • Middle Name (Optional)
        • Last Name (Required)
        • Educational(Degree) Prefixes (Optional) e.g: click here .
        Buttons
        • Update Updates the record and redirects to the Manage Authors Page
        • Update and continue editing Updates the record but retains to the current page
        • Back Redirects to the Manage Authors Page
    7. Delete Author Removes/Delete Author details permanently.
      Note!

      Please take caution before performing this action

      1. To delete a record on the table, click the delete button of the record you want to remove.
      2. The user will be redirected to delete confirmation page. Once the user confirms, the record will be deleted immediately.

Dashboard

Important Note: The dashboard page contains the detailed summary or statistics of data that is stored on the database system.

  1. To access the Dashboard, Go to the Navigation Menu on the top right side of the page and select the Dashboard Menu.
  2. Once the user clicked the dashboard menu, it will be redirected to dashboard page.
    • Authors Contains the total authors enrolled on the system.
    • Researches Contains the total researches enrolled on the system.
    • Total Visitors Contains the total number of visitors on every researches visited on the system.

Logging Out

Important Note: It is important for user to close or Log-out their sessions before closing the browsers.

  1. Go to the profile menu on the top-right corner of the page.
  2. Click Log-out
